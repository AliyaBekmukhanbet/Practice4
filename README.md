# Practice4

Контрольные вопросы

1. Чем отличаются типы памяти в CUDA и в каких случаях их
использовать?
| Тип памяти | Скорость      | Объём         | Видимость | Когда использовать   |
| ---------- | ------------- | ------------- | --------- | -------------------- |
| Регистры   | Очень высокая | Очень малый   | Поток     | Локальные переменные |
| Local      | Низкая        | Большой       | Поток     | Избегать             |
| Shared     | Высокая       | Малый         | Блок      | Общие данные блока   |
| Global     | Низкая        | Очень большой | Все       | Основные данные      |
| Constant   | Высокая       | Малый         | Все       | Константы            |
| Texture    | Высокая       | Средний       | Все       | Изображения          |
| Unified    | Средняя       | Большой       | Все       | Простота разработки  |
2. Как использование разделяемой памяти влияет на производительность?
Использование разделяемой памяти может значительно повысить производительность, поскольку она обеспечивает намного более быстрый доступ к данным, чем глобальная память, и позволяет сократить количество медленных обращений к ней.
3. Доступ и как его обеспечить?
Под доступом к памяти в CUDA понимается способ, которым потоки обращаются к данным в памяти GPU. От этого напрямую зависит производительность программы.
4. Какие сложности возникают при работе с большим объемом данных на
GPU?
Основные сложности работы с большими данными на GPU связаны с ограниченной памятью, высокой стоимостью передачи данных, особенностями доступа к памяти и необходимостью тщательной оптимизации ресурсов.
5. Почему важно минимизировать доступ к глобальной памяти?
Минимизация доступа к глобальной памяти позволяет скрыть задержки, увеличить пропускную способность и раскрыть вычислительный потенциал GPU.
6. Как использовать профилирование для анализа производительности
CUDA-программ?
Профилирование CUDA-программ позволяет выявить узкие места, оценить эффективность использования ресурсов GPU и направленно оптимизировать доступ к памяти и конфигурацию потоков.
